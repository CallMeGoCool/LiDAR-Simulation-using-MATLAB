# LiDAR-Simulation-using-MATLAB
A matlab simulation of working of LiDAR, where it detects an object and gives its approximate coordinates.

Here, we are simulating a LiDAR's working, where it takes multiple scans of slices, with each slice referring to a complete rotation around its axis such that it takes a scan of the XY plane. As such by stacking multiple scans of the XY plane across a variable Z-axis, it becomes easier to get a 3D image of the surroundings, which can be used to calculate the distance of an object wrt an origin.
